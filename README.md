# Global Superstore EDA Project

Professional Exploratory Data Analysis (EDA) project on the `Global_Superstore.csv` retail dataset using Python.

## Project Overview

This project analyzes sales transactions to identify:
- sales and profit trends
- category and regional performance
- discount impact on profitability
- monthly and yearly seasonality patterns

The analysis is designed to be portfolio-ready for GitHub and resume use.

## Project Files

- `Global_Superstore.csv` - source dataset
- `Global_Superstore_EDA.ipynb` - full notebook version
- `Global_Superstore_EDA.py` - script version of complete EDA
- `eda_outputs/` - saved chart images generated by script

## Setup Instructions

1. Create and activate virtual environment (recommended):

```bash
python -m venv .venv
```

Windows:

```bash
.venv\Scripts\activate
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

## Run the Project

Run notebook:

```bash
jupyter notebook Global_Superstore_EDA.ipynb
```

Run Python script:

```bash
python Global_Superstore_EDA.py
```

## Output Graphs

When you run the script, charts are saved automatically in `eda_outputs/`:
- `01_univariate_distributions.png`
- `02_bivariate_analysis.png`
- `03_correlation_heatmap.png`
- `04_time_based_analysis.png`
